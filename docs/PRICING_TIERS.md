# Agent Factory Platform: Pricing Tiers

## Overview

Agent Factory offers flexible pricing to serve developers, startups, agencies, and enterprises. Our pricing model combines SaaS tiers with usage-based options and marketplace revenue sharing.

---

## SaaS Tiers

### Free Tier — "Explorer"

**Target**: Developers exploring the platform, students, hobbyists

**Included:**
- 1 agent
- 1,000 agent requests/month
- Community tools only (web search, file I/O, calculator)
- Local registry access
- CLI access
- Community support (Discord, GitHub Discussions)
- Basic documentation

**Limitations:**
- No API access
- No Blueprint marketplace access
- No custom tool integrations
- No advanced features (scaling, monitoring)

**Use Case**: Learning, prototyping, personal projects

**Upgrade Path**: → Starter ($49/mo)

---

### Starter — "$49/month"

**Target**: Solo founders, small projects, early-stage startups

**Included:**
- 5 agents
- 10,000 agent requests/month
- All community tools
- Basic Blueprint marketplace access (free Blueprints only)
- REST API access (rate-limited: 100 requests/hour)
- Basic monitoring (request logs, error tracking)
- Email support (48-hour response)
- Blueprint creation tools

**Add-ons Available:**
- Additional requests: $0.01 per 1,000 requests
- Tool packs: $29-99/month each
- Premium Blueprints: One-time purchase ($0-$999)

**Use Case**: MVP development, small SaaS products, personal automation

**Upgrade Path**: → Pro ($199/mo)

**ROI Example**: 
- Build customer support bot → Save $2,000/month on support tools
- Launch in 2 weeks vs 2 months → 4x faster time to market

---

### Pro — "$199/month"

**Target**: Growing startups, agencies, mid-market companies

**Included:**
- 20 agents
- 100,000 agent requests/month
- All community tools + 2 tool packs included
- Full Blueprint marketplace access (all Blueprints)
- REST API access (1,000 requests/hour)
- Advanced monitoring (metrics dashboard, alerts, tracing)
- Priority email support (24-hour response)
- Blueprint creation and publishing tools
- Revenue share: 70% on Blueprint sales (vs 30% platform fee)
- Custom tool development support

**Add-ons Available:**
- Additional agents: $10/month per agent
- Additional requests: $0.008 per 1,000 requests (volume discount)
- Additional tool packs: $29-99/month each
- White-label options: $99/month

**Use Case**: Production SaaS products, agency client work, internal automation

**Upgrade Path**: → Enterprise ($999+/mo)

**ROI Example**:
- Agency: Deliver 5 client projects/month → $15K revenue, $3K costs → 80% margin
- Startup: Launch 3 AI features → Increase user engagement 30%

## Usage-Based API Pricing

**Target**: Developers building custom solutions, high-volume users, one-off projects

### Pricing Structure

- **Agent Requests**: $0.01 per request (first 10,000 free/month)
- **Tool Executions**: $0.10 per execution (first 1,000 free/month)
- **Workflow Executions**: $0.50 per execution (first 100 free/month)
- **Storage** (memory/sessions): $0.10 per GB/month (first 10 GB free)

### Volume Discounts

- **100K+ requests/month**: 20% discount ($0.008 per request)
- **500K+ requests/month**: 30% discount ($0.007 per request)
- **1M+ requests/month**: 40% discount ($0.006 per request)
- **Custom pricing**: Available for 10M+ requests/month

### Use Cases

- **High-Volume Applications**: Chatbots, content generation, data processing
- **One-Off Projects**: Prototypes, experiments, temporary automations
- **Custom Integrations**: Building on top of Agent Factory API

**Example Costs**:
- 50K requests/month: $500/month
- 200K requests/month: $1,600/month (with 20% discount)
- 1M requests/month: $6,000/month (with 40% discount)

**Free Blueprints**: Community-created, open-source
- No cost to install
- May require API keys for integrations

**Premium Blueprints**: One-time purchase or subscription

**Enterprise Blueprints**: Custom pricing
- Negotiated with creator
- May include white-label options
- Custom support available

**Platform Commission**: 30% (standard)

**Creator Earnings Example**:
- Blueprint sells for $99 one-time
- Standard: Creator earns $69.30, Platform earns $29.70
- Pro tier: Creator earns $79.20, Platform earns $19.80

**Payout**: Monthly, minimum $50

## Pricing Strategy Rationale

### Free Tier
- **Goal**: Remove friction, build user base
- **Strategy**: "Try before you buy", viral growth
- **Risk**: High support costs, low conversion
- **Mitigation**: Community support, clear upgrade prompts

### Starter Tier
- **Goal**: Convert free users, low barrier to entry
- **Strategy**: Price point for solo founders ($49 = 1 coffee/day)
- **Risk**: Low revenue per user
- **Mitigation**: Volume, upsell to Pro

### Pro Tier
- **Goal**: Main revenue driver, target market
- **Strategy**: Sweet spot for startups/agencies ($199 = affordable but valuable)
- **Risk**: Competition from free alternatives
- **Mitigation**: Value-add features, marketplace access

### Enterprise Tier
- **Goal**: High-value contracts, predictable revenue
- **Strategy**: Custom pricing, value-based
- **Risk**: Long sales cycles
- **Mitigation**: Self-service → Pro → Enterprise path
